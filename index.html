Installation and Setup

  • Install the Anaconda distribution of Python 2.7x.
  • Install Git and create a GitHub account.
  • Once you receive an email invitation from Slack, join our "SYD_DAT_4 team" and add your photo!

Resources

  • PEP 8 - Style Guide for Python

Class 1: Introduction / Expectations / Git Intro

  • Introduction to General Assembly
  • Course overview: our philosophy and expectations (slides)
  • Tools: check for proper setup of Git, Anaconda, overview of Slack

Homework:

  • Resolve any installation issues before next class.
  • Make sure you have a github profile and created a repo called "SYD_DAT_3"
  • Clone the class repo (this one!)
  • Review this code for a recap of some Python basics.

Optional:

  • Read Analyzing the Analyzers for a useful look at the different types of data scientists.
  • Read about some Markdown Techniques
  • Write a summary of 2 chapters of The Data Science Handbook in Markdown and submit a change request in the Lab Directory
